Migration Pattern

Example of migrating from current CSS to Tailwind

Before (Current CSS)
<button class="btn btn-primary">Click me</button> <div class="post-card">...</div>
After (Tailwind)
<button class="tw-btn-primary">Click me</button> <div class="tw-post-card">...</div>
During Migration (Both work)
<!-- Old styles continue to work --> <button class="btn btn-primary">Old Button</button> <!-- New Tailwind styles with tw- prefix --> <button class="tw-btn-primary">New Button</button>
